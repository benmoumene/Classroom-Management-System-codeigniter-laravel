Hosted Services

If you'd just like to give Queso a test spin, you can sign up for a free hosted account. To create an instructor account and course, just go to queso.cc/register/instructor

Host It Yourself?

If you're the type of person that likes having everything on your own server, just download the files and stick it on your server. Queso is built with Laravel, so you should migrate the tables to setup the MySQL database. In the future, there will be better documentation for this.
